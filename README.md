# Object-Oriented-Programming-Sebastian-Villegas
TC1030 Portfolio Project:
This project is about an investment portfolio that manages different types of financial assets. To organize the assets owned by an investor, a class structure is built that separates them into Stocks, Cryptocurrencies and Bonds. Each type of asset is valued differently: stocks depend on the market price of the company, cryptocurrencies on the real-time price of the coin, and bonds on the face value plus the accumulated interest rate.
Functionality:
The program allows adding new assets to the portfolio and checking the total value of the investment, either for the entire portfolio or filtering by asset type: stocks cryptocurrencies bonds.
Considerations:
The program only runs on the console and is made with c++ standard so it runs on all operating systems
compile with: "g++ asset.h portfolio.h main.cpp"
run on linux: "/a.out"
run on windows: "a.exe"
